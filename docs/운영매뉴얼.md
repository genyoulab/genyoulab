# GenYou Lab 운영 매뉴얼

본 매뉴얼은 GenYou Lab 프론트엔드 프로젝트의 유지보수 및 운영을 담당하는 작업자를 위한 가이드 문서입니다.

## 1. 신규 제품 페이지 추가 방법
1. 기존의 제품 페이지(예: `reset-mist.html`)를 복사하여 새로운 HTML 파일(예: `new-product.html`)을 생성합니다.
2. `<title>`, `og:title`, 그리고 내부 텍스트 콘텐츠 및 이미지 에셋 경로를 신규 제품에 맞게 변경합니다.
3. **스마트스토어 URL 매핑**: 
   - `src/config/smartstore.js` 파일을 열어 `SMARTSTORE_URLS` 객체에 새로운 `product_id`와 UTM이 적용된 스마트스토어 URL을 추가합니다.
   - 예: `'new_product': 'https://smartstore.naver.com/genyoulab/products/999999?utm_source=genyoulab&utm_medium=website&utm_campaign=New_Product'`
4. **구매 버튼 연결**: 
   - 신규 HTML 파일 내의 구매 버튼 `onclick` 이벤트에 해당 ID를 부여합니다.
   - 예: `onclick="goToSmartStore('new_product')"`

## 2. 소셜 API 키 갱신 방법
소셜 로그인 키 만료, 도메인 변경 혹은 앱 삭제 후 재발급 시 아래 파일의 변수를 업데이트해야 합니다.
- **파일 경로**: `src/config/auth.js`
- **카카오**:
  - `KAKAO_APP_KEY` 변수의 값을 [카카오 디벨로퍼스 > 내 애플리케이션 > 앱 키 > JavaScript 키] 항목의 값으로 교체합니다.
- **네이버**:
  - `NAVER_CLIENT_ID` 변수의 값을 [네이버 개발자 센터 > Application > 내 애플리케이션 > Client ID] 항목의 값으로 교체합니다.
  - 도메인이 변경된 경우 `NAVER_CALLBACK_URL` 도 수정해야 합니다.

## 3. 스마트스토어 URL 변경 및 UTM 갱신 방법
제품의 네이버 스마트스토어 등록 주소가 바뀌거나, 마케팅 캠페인(`utm_campaign`)을 시즌별로 교체할 때 사용합니다.
- **파일 경로**: `src/config/smartstore.js`
- 해당 객체의 값을 찾아 URL 전체를 원하는 타겟 URL로 덮어씁니다.
- 변경 후 `firebase deploy` 명령어를 통해 배포해야 실시간으로 반영됩니다.

## 4. GA4 전환 퍼널 확인 방법
본 프로젝트는 구매 의향 및 브릿지 모달 뷰에 대해 GA4 커스텀 이벤트 전송 로직이 구현되어 있습니다. (`gtag('event', ...)` 방식)
1. **Google Analytics 대시보드** 접속 (속성 ID: `G-DVR7NZR8MW`)
2. **[참여도] -> [이벤트]** 메뉴로 진입.
3. 수집된 이벤트 목록 중 다음 세 가지를 확인합니다:
   - `purchase_intent`: 유저가 '네이버에서 구매하기' 버튼을 최초 클릭한 횟수
   - `bridge_modal_shown`: 브릿지 모달이 정상적으로 띄워진 횟수
   - `smartstore_redirect`: 모달 로딩(0.8초) 완료 후 실제 스마트스토어로 넘어간 횟수
4. 위 세 수치의 차이를 비교하여 로딩 중 이탈률을 모니터링할 수 있습니다.

## 5. `rules.md` (코딩 컨벤션) 업데이트 기준
프로젝트의 무결성을 유지하기 위해 코어 디자인 시스템 및 정책을 관리하는 `rules.md` 파일은 엄격히 관리되어야 합니다.
- **디자인 토큰 변경**: 컬러 코드, 폰트 스케일 등 브랜드 아이덴티티가 변경될 경우 반드시 사전에 기획팀과 합의 후 `rules.md`의 [Section 1]을 먼저 업데이트한 뒤, 실제 `tokens.css` 코드를 반영해야 합니다.
- **컴포넌트 추가**: 새로운 유형의 모달이나 내비게이션 바가 추가될 경우, 네움오피즘/글래스모피즘 등 규칙에 부합하는지 확인 후 [Section 2]에 명세서를 추가합니다.
- **금지어 업데이트**: 브랜드 카피라이팅 기조인 "내버려 두는 것의 미학"에 위배되는 새로운 금지어 리스트가 생기면 [Section 2-1]에 지속적으로 추가하고, 전체 검색(`grep`)을 통해 사이트 내에 잔존하지 않는지 주기적으로 감시합니다.
